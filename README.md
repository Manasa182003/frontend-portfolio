# Portfolio Website

## Project Description
This is a personal portfolio website built using **HTML, CSS, and JavaScript**.  
It showcases my projects, provides an about section, and allows visitors to interact with the website through simple JavaScript functionality.  
This project demonstrates core front-end development skills suitable for fresher web developers.

---

## Features
- **Responsive Navigation Bar** – Quickly access About, Projects, and Contact sections.  
- **About Section** – Short introduction about myself.  
- **Projects Section** – Highlights the projects I have developed.  
- **Contact Section** – Includes a JavaScript button for interactive messaging.  
- **Clean UI Design** – Styled using CSS for readability and aesthetics.  

---

## Technologies Used
- **HTML5** – Structure of the website.  
- **CSS3** – Styling and responsive design.  
- **JavaScript** – Interactive features and DOM manipulation.
